# 選手権アプリ トンマナ・デザインガイドライン

## 1. ブランドコンセプト

### 1.1 コアバリュー
**「みんなで楽しむ大喜利プラットフォーム」**

- **遊び心**: 肩肘張らず、気軽に参加できる楽しさ
- **共創**: 主催者も参加者も、みんなで作り上げる
- **発見**: 思わず「そうそう！」と頷く回答や、「なるほど！」と唸る回答との出会い

### 1.2 ペルソナ
- **年齢層**: 20代〜40代
- **SNS慣れしている**: TwitterやInstagramなどで日常的に投稿している
- **ユーモアを楽しむ**: 面白い投稿やあるあるネタが好き
- **クリエイティブ**: 自分なりの視点で表現したい

## 2. ビジュアルアイデンティティ

### 2.1 カラーパレット

#### メインカラー
- **プライマリ**: `#FF6B35` (コーラルオレンジ)
  - 用途: メインボタン、アクション、強調
  - イメージ: 元気、遊び心、ポジティブ
  
- **セカンダリ**: `#4ECDC4` (ターコイズ)
  - 用途: サブボタン、アクセント、選定中のステータス
  - イメージ: 爽やか、クリエイティブ

#### サポートカラー
- **募集中**: `#FFD93D` (明るいイエロー) - 活気、募集の呼びかけ
- **結果発表**: `#6C5CE7` (ソフトパープル) - 特別感、祝祭
- **受賞**: `#FFB800` (ゴールド) - 栄誉、達成

#### ニュートラルカラー
- **背景**: `#F8F9FA` (ライトグレー) - 清潔感
- **テキスト**: `#2D3436` (ダークグレー) - 読みやすさ
- **境界線**: `#DFE6E9` (ミドルグレー)

### 2.2 タイポグラフィ

#### フォントファミリー
- **見出し・タイトル**: Noto Sans JP Bold
  - 選手権タイトル、セクション見出し
  - しっかりした印象、視認性
  
- **本文**: Noto Sans JP Regular/Medium
  - 説明文、回答テキスト、コメント
  - 読みやすさ重視

#### フォントサイズ（参考）
```
見出し1（画面タイトル）: 24sp
見出し2（セクション）: 20sp
見出し3（カード内タイトル）: 18sp
本文: 16sp
キャプション: 14sp
補足: 12sp
```

### 2.3 アイコン・イラスト
- **スタイル**: シンプル、ラインベース（Lucide Icons など）
- **カラー**: 基本はグレー、アクション時にプライマリカラー
- **サイズ**: 24dp 基準（タップ領域は48dp確保）

### 2.4 写真・画像
- **回答画像**: ユーザー投稿（9:16推奨）
- **角丸**: 8dp（カードの場合は12dp）
- **プレースホルダー**: グラデーション背景 + アイコン

## 5. アニメーション・インタラクション

### 5.1 基本原則
- **快適な速度**: 200-300ms（速すぎず遅すぎず）
- **意味のある動き**: 目的のない装飾的なアニメーションは避ける
- **一貫性**: 同じ操作には同じアニメーション

### 5.2 主要なアニメーション

#### いいねボタン
```
タップ時:
  - スケール: 1.0 → 1.2 → 1.0（150ms）
  - 色変化: グレー → レッド
  - パーティクルエフェクト（小さなハート）
```

#### カード表示
```
リスト読み込み時:
  - フェードイン + スライドアップ（200ms）
  - 各カードを50msずつ遅延してスタガー表示
```

#### 画面遷移
```
Forward:
  - スライドイン（右から左、300ms）
  
Back:
  - スライドアウト（左から右、300ms）
```

#### ボタンタップ
```
リップルエフェクト:
  - 中心から外へ波紋（200ms）
  - 色: プライマリカラーの20%透明度
```

## 9. アクセシビリティ

### 9.1 カラーコントラスト
- テキストと背景のコントラスト比: 最低4.5:1（WCAG AA準拠）
- 重要なアクションボタン: 7:1以上

### 9.2 タップターゲット
- 最小サイズ: 48dp x 48dp
- 間隔: 8dp以上

### 9.3 フォントサイズ
- 本文: 最小16sp（読みやすさ重視）
- スケーラビリティ: システムフォントサイズ設定に対応
